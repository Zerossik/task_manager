# Правила по написанию кода.

## Общие правила:

- use TypeScript.

## Material UI

- используй компоненты из библиотеки MaterialUI.

## Практика импортов и Public API

- **Абсолютные импорты:** При перемещении между слоями используй только алиасы (например, `@src/*`). Никаких относительных путей вида `../../../../shared`.
- **Изоляция (Public API):** Импортировать компоненты из `shared/ui` или `pages` можно только через их корневые файлы `index.ts`. Глубокие импорты (например, `src/shared/ui/Button/Button.tsx`) запрещены.
